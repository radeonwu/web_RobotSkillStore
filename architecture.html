Architecture & RSP Protocol

体系结构与 RSP 协议

Robot Skill Store is built on the Robot Skill Protocol (RSP) – a schema-driven, middleware-agnostic skill API, semantically aligned with RCP's READ/WRITE/EXECUTE/SUBSCRIBE model.

Robot Skill Store 底层采用 Robot Skill Protocol（RSP）作为统一协议， 使用 Schema 驱动的方式对技能进行定义，并在语义上对齐 RCP 的 READ/WRITE/EXECUTE/SUBSCRIBE 模型。

RSP API surface

RSP 接口概览

Management APIs

管理接口

  • Skill.register
  • Skill.update
  • Skill.list
  • Skill.describe
  • 技能注册：Skill.register
  • 技能更新：Skill.update
  • 技能列表：Skill.list
  • 技能描述：Skill.describe

Execution APIs

执行接口

  • Skill.call (EXECUTE)
  • Skill.monitor (READ)
  • Skill.cancel (EXECUTE/WRITE)
  • Skill.subscribeEvents (SUBSCRIBE)
  • 技能调用：Skill.call（执行）
  • 状态查询：Skill.monitor（读取）
  • 取消任务：Skill.cancel（执行/写入）
  • 事件订阅：Skill.subscribeEvents（订阅）
