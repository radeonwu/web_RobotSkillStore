Platform overview

平台概览

Robot Skill Store provides a full-stack platform for discovering, validating, and running robot skills.

Robot Skill Store 提供一整套用于发现、验证和运行机器人技能的平台能力。

Skill Store

技能商店

Browse and install skills such as Pick & Place, screw tightening, connector insertion, and visual inspection.

浏览并安装各类技能，例如抓取放置、拧紧、插接、视觉检测等模块化技能。

RSP Protocol

RSP 统一协议

Unified skill API: Skill.register / Skill.call / Skill.monitor / Skill.cancel. Compatible with RCP semantics.

提供统一的技能 API：Skill.register / Skill.call / Skill.monitor / Skill.cancel， 并在语义上兼容 RCP 的 READ / WRITE / EXECUTE / SUBSCRIBE 模型。

Safety & Validation

安全与验证

Built-in simulation, trajectory checks, and safety proxies ensure safe deployment before production use.

通过内置仿真、轨迹检查和安全代理，在真实产线使用前完成安全验证。
