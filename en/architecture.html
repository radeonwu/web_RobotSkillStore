Architecture

Platform-Centered, Protocol-Enforced

Robot Skill Store is built around a deliberate choice: the platform is the center of the system. RSS provides execution, evaluation, and governance — and it enforces a shared protocol (RSP) so tools, simulators, robots, and agents can interoperate without redefining semantics.

RSP as the Architectural Backbone

The Robot Skill Protocol (RSP) defines how skills are described, discovered, constrained during execution, and how outcomes are reported. Because these interactions are standardized:

  • skills can move across systems
  • agents can reason independently of implementation
  • execution environments remain interchangeable

RSP is not an internal API. It is the shared contract that enables coordination across the ecosystem.

Decoupling Capability from Implementation

Traditional architectures tightly bind behavior logic, hardware configuration, and environment assumptions. Robot Skill Store explicitly separates:

  • what a skill means
  • how it is implemented
  • where it is executed

This separation allows independent evolution of skills and hardware, consistent behavior across environments, and validation before deployment.

Simulation and Reality as a Single Loop

Simulation is not a testing afterthought. In Robot Skill Store, simulation is part of execution semantics. A skill must declare its environmental assumptions; results are comparable across simulation and real robots.

Closed Loop

define → simulate → validate → execute → feedback

The same skill specification governs all stages.

Where Safety Fits in the Architecture

Safety is not an external layer. It is embedded in skill definitions, execution constraints, and runtime enforcement. Placing safety in skill contracts and runtime policy gates (protocol-enforced) reduces reliance on ad-hoc checks and makes behavior more predictable across environments.

Extensible by Design

Because RSS enforces a stable protocol backbone (RSP), new simulators, robots, and agents can be added without redefining the entire system. This extensibility is what makes Robot Skill Store a foundation for long-term ecosystem growth.
