RSS Skills Summary

In RSS, skills are managed assets with lifecycle: spec → build → benchmark → publish → run → audit.

Skill as an asset
  • Immutable versions with provenance.
  • Compatibility and constraints declared.
  • Benchmarks define readiness.
Run as the unit of truth
  • Every run binds (skill version + profile + policy).
  • Artifacts are mandatory (evidence).
  • Aggregations power public dashboards.
