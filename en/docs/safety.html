Skill-level Safety

Safety is enforced at the skill boundary

Overview Concepts Skill Spec Schema Execution Safety Demo Mapping

Safety layers

  1. Declared boundaries: constraints in the skill spec
  2. Runtime enforcement: monitor and stop when violated
  3. Agent policies: selection-time risk controls
  4. Evidence & audit: post-run review and accountability

Fail-fast, evidence-first

If safety constraints cannot be enforced, execution must not proceed. Fail fast and produce evidence.

Relation to the website Safety page

The website explains the stance. This page defines enforceable mechanisms and expected artifacts.
