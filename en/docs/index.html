RSP Docs

Robot Skill Protocol v0.3

RSP is the protocol backbone of Robot Skill Store. It standardizes how skills are described, discovered, validated, executed, and audited — consistently across simulators and real robots.

Reading order

  1. Concepts: Skill / Agent / Runtime / Evidence
  2. Skill Spec: canonical fields and semantics
  3. Execution: binding, enforcement, outcomes, evidence
  4. Safety: skill-level stance and mechanisms
  5. Demo Mapping: UR5 reference execution model

Design stance

  • Specs are first-class: agents reason over contracts, not code.
  • Runtime enforces constraints; safety is not a best-effort add-on.
  • Evidence is required for auditability and iteration.
