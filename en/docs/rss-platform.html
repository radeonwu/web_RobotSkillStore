RSS Platform (Summary)

What the RSS product provides: registry, versioned runs, evaluation, and runtime governance.

Back to Docs HubArchitectureSafetySkillsAgents

Core capabilities

  • Skill registry: publish skills as deployable assets with metadata, versions, and compatibility.
  • Run orchestration: every execution becomes a Run bound to a specific skill version.
  • Evaluation: benchmarks + reports produce objective, comparable performance signals.
  • Governance: policy gates enforce approval, safety constraints, and authorization at runtime.
  • Evidence: artifacts (logs, video, metrics) make outcomes auditable and reproducible.
