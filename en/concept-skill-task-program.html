Skill ≠ Task ≠ Program

Abstraction layers

  • Program: implementation code (many variants)
  • Task: a goal in a scenario (context-dependent)
  • Skill: reusable capability with explicit interface + constraints

RSS focuses on skills because they can be validated and reused across scenarios once certified.
