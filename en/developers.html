Minimal example
# Pseudo-interface Skill.register(manifest, endpoint, verification_artifacts) result = Skill.call( name="pick_and_place", inputs={...}, mode="async", timeout_s=30 )

Your implementation can wrap ROS2, MoveIt2, Isaac, or proprietary stacks.

Recommended workflow

1) Author

Define the manifest: IO schema, constraints, preconditions, and telemetry hooks.

2) Validate

Run sim-first tests, generate verification artifacts, and attach them to the skill.

3) Publish

Push to a store with versioning, licensing, and trust level metadata.

4) Deploy

Operators choose policies, runtimes, and robots that match capability profiles.

5) Observe

Monitor standardized logs/metrics/traces; compare performance across sites.

6) Improve

Ship updates safely with backward compatibility and controlled rollouts.
