What makes a “skill” in RSP

Skills are bounded, contract-defined, agent-callable capabilities — portable across robots that match capability constraints.

Manifest

Machine-readable metadata: IO schema, preconditions, constraints, version, licensing, telemetry hooks.

Runtime

Execution sandbox with safety guards, resource limits, monitoring, and standardized error semantics.

Store

Discovery, trust signals, verification reports, licensing, distribution, and updates.

Typical deployment patterns

On-robot

Skills run on edge controllers for low latency and robust operation.

Edge cluster

Fleet-level orchestration and monitoring in a local cluster near production.

Cloud

Central store, CI/CD, evaluation, simulation at scale, and analytics.
