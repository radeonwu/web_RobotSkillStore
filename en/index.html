Built on Evidence, Not Promises

Every RSS skill is backed by immutable execution records, not just a README. Governance state is computed continuously from real data.

📝

Immutable Run Records

Every execution creates a Run bound to skill version, robot profile, and parameter set — immutable after creation.

⚙

Computed Governance

ACTIVE / DEGRADED / QUARANTINED is computed from rolling evidence windows — success rate, violations, regression — not assigned by hand.

🔎

Full Audit Trail

Logs, videos, telemetry, and certification snapshots are linked to each Run — fully auditable and reproducible.
