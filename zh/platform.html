平台能力

🔍
技能注册表
接受经 JSON Schema 校验的结构化技能规格，分配稳定 ID，技能立即在公开目录中可被检索。
▶
运行管理
每次执行创建一条不可篡改的 Run 记录——与技能版本、机器人配置、参数集绑定。结果、约束与原因全部索引。
📦
制品存储
日志、视频、遥测数据存储（本地磁盘或 S3/MinIO）并与对应 Run 绑定，上传后不可修改，可通过证据查看器访问。
⚙
治理引擎
从滚动证据窗口计算 ACTIVE / DEGRADED / QUARANTINED——成功率、违规率、回归信号。无人工指派。
🏆
认证
证据满足策略阈值后颁发治理快照，技能状态晋升为 CERTIFIED，连同证据链路一起公开发布。
🔒
认证与访问控制
公开读取路径（/public/v1/skills）无需凭据。写入与管理操作需要 API Key。匿名读取模式可按部署配置。

流程总览

从技能注册到公开认证——六个阶段，全部有证据门控。

1) 注册
提交技能规格 · Schema 校验 · 分配技能 ID
➜
2) 执行
提交运行记录（仿真 / HIL / 真实机器人）· 记录结果与约束
➜
3) 采集证据
挂载制品（日志、视频、遥测）· 不可篡改存储
➜
6) 发布
已认证技能在公开目录可见 · 网站嵌入实时数据
➜
5) 认证
颁发治理快照 · 生命周期晋升为 CERTIFIED
➜
4) 评估
滚动窗口 · 成功率、违规率、回归信号

证据查看器

内置证据查看器让任何人都能浏览运行记录、检查制品、审阅治理快照——无需任何代码。

  • 浏览任意技能版本的所有运行——按结果、机器人、日期筛选。
  • 查看附属制品：视频回放、日志检查、遥测图表。
  • 检查治理快照：通过什么、什么触发了 DEGRADED，以及原因。
