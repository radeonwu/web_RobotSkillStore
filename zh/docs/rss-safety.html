RSS 安全摘要

安全不是“最佳努力”，而是 运行时强制：所有动作都必须通过策略门禁，并留下可审计证据。

关键机制
  • Policy Gates：授权/审批/范围限制不可绕过。
  • Run 证据：日志、视频、指标、制品可追溯。
  • 版本绑定：结果绑定 Skill Version + Profile。
  • 最小暴露：网站只展示公开聚合，不泄露项目细节。
你应当如何表述
  • ✅ “平台强制安全门禁与审计记录”
  • ✅ “所有执行都是可追溯的 Run”
  • ❌ “一键控制机器人 / 智能体直接执行”
