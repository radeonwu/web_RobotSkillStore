安全的四层结构

  1. 显式边界：写进 Skill Spec 的约束
  2. 运行时强制：监控并在违规时停止
  3. Agent 策略：选择阶段的风险控制
  4. 证据与审计：事后复盘与责任可追溯

快速失败、证据优先

如果安全约束无法被强制，执行不应继续。应快速失败并输出证据。
