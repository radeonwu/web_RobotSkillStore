RSS · 技能摘要

技能不是“代码片段”，而是可部署、可评测、可治理的系统能力资产。

Skill（能力）

  • 对外合同：输入/输出/前置条件/失败模式。
  • 可组合：能被工作流与智能体调用。

Skill Version（版本）

  • 不可变：每次运行绑定到明确版本。
  • 可回滚：升级与回退都可审计。

评测与证据

  • 以 Benchmark 判定成功，不靠主观 Demo。
  • 每次 Run 生成证据与制品（日志/视频/指标）。
