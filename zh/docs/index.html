RSP 文档

Robot Skill Protocol v0.3

RSP 是 Robot Skill Store 的协议主干，用于统一技能的描述、发现、验证、执行与审计，确保仿真与真实机器人之间具有一致的语义与约束。

总览 概念 技能规格 Schema 执行语义 安全模型 Demo 映射

推荐阅读顺序

  1. 核心概念：Skill / Agent / Runtime / Evidence
  2. 技能规格：规范字段与语义
  3. 执行语义：绑定、强制、结果与证据
  4. 安全模型：技能级安全立场与机制
  5. Demo 映射：UR5 参考执行模型

设计立场

  • 规格是一等公民：Agent 依赖契约，而不是依赖代码细节。
  • 运行时强制约束：安全不是“尽力而为”的外挂层。
  • 证据必须可用：审计、评测与迭代都依赖 Evidence。
