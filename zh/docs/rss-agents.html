RSS 智能体摘要

智能体是 RSS 平台内部的决策层：提出 Run 请求，平台负责强制与审计。

智能体的角色

  • 选择：从可用技能/版本中挑选候选（受策略约束）。
  • 编排：把多个技能组合成工作流（含分支、重试、升级处理）。
  • 依据证据：使用 Benchmark/Run 结果做决策，而不是 UI 观察。

不可妥协规则

Agents decide what to try. RSS enforces what is allowed to happen.
