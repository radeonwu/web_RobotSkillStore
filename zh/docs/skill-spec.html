技能规格（Skill Spec）

Agent 可读、Runtime 可强制的能力契约

总览 概念 技能规格 Schema 执行语义 安全模型 Demo 映射

为什么规格不是代码

实现会变，契约必须稳定。RSP 将技能规格视为能力契约：既能被 Agent 理解，也能被运行时强制。

四个不可缺部分

  • 意图（Intent）：技能要达成什么（目标语义）
  • 上下文（Context）：对机器人与环境的假设
  • 约束（Constraints）：执行过程中必须被强制的边界
  • 结果（Outcomes）：可观测的成功/失败与证据要求

谁来强制

运行时负责绑定与强制。如果某个约束无法强制，执行应快速失败并输出证据。
