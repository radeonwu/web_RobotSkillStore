开发者

快速开始：运行你的第一个技能

Robot Skill Store 设计为渐进式引入：无需替换现有控制栈，只需在能力层引入 RSP 的规范与运行时绑定。

你需要准备什么

  • 机器人或仿真器（UR5 / Gazebo / Isaac Sim 等）
  • 基础运行环境（ROS / Python 等）
  • 一个遵循 RSP 的技能定义

最小工作流

  1. 定义技能：明确意图、约束、上下文与可观测结果
  2. 注册技能：发布到技能注册表（Skill Store）
  3. 选择技能：Agent 基于目标、上下文与策略做决策
  4. 执行技能：运行时绑定参数并强制约束

执行与反馈

执行过程中，参数被动态绑定、约束被持续强制、结果被结构化记录。反馈不是日志附属物，而是生命周期的一部分，可用于后续选择与改进。
