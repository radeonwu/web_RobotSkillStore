机器人技能是一种可被验证、认证并在不同部署场景中复用的执行原语。

可在真实机器人上执行 · 确定性行为（在声明约束范围内）· 显式前置条件、安全限制与 I/O Schema · 可通过证据产物和指标验证

✕ 不只是…

  • 动作脚本或轨迹
  • 一次性演示或原型
  • 硬编码、不可迁移的程序

✓ 技能是…

  • 可调用、可评估、可迁移
  • 由显式契约约束
  • 由证据认证，而非凭声誉
I/O 语义
明确的输入/输出 Schema 与约束定义了技能接口，使其可跨系统组合。
实现自由
实现可采用规则、RL、VLA、世界模型或混合方式——接口契约保持不变。
可量化质量
成功率、适用域与风险边界是一等属性，而非事后补充。
Agent 就绪编排
技能可调度、可组合、可替换，支持 Agent 驱动的安全工作流。

执行成本与风险越高，验证与治理越不可缺。RSS 让机器人技能具备可移植性与可信度。
