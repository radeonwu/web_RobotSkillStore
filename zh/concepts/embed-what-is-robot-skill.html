什么是 Robot Skill？

RSS 中的 Robot Skill 是一种可确定执行的机器人能力单元，在明确约束下控制真实机器人，并能够被验证、认证与复用。

定义

Robot Skill 是物理执行层的能力基元，具备可验证、可认证、可复用的工程属性。

  • 可在真实机器人上执行
  • 确定性行为（在已声明约束内）
  • 显式前置条件 / 安全边界 / I/O Schema
  • 可通过证据与指标进行验证

核心思想（What）- Robot Skill 的重新定义

  • Skill ≠ 动作脚本 / Trajectory / Demo
  • Skill = 可调用、可评估、可迁移的能力单元

Skill 必须具备

  • 明确的输入 / 输出语义（I/O schema 与约束）
  • 内部实现可多样：规则 / RL / VLA / World Model / 混合
  • 可量化的质量属性：成功率、适用域、风险边界
  • 可被 Agent 调度：可组合、可替换、可编排

为什么 RSS 要强调 Skill

物理执行的成本与风险远高于软件调用，因此必须引入验证、证据与治理机制。RSS 的目标是让技能可迁移、可信、可规模化复用。
