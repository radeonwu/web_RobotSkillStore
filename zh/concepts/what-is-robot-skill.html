什么是 Robot Skill？

RSS 中的 Robot Skill 是一种可确定执行的机器人能力单元，在明确约束下控制真实机器人，并能够被验证、认证与复用。

定义

Robot Skill 是物理执行层的能力基元，具备可验证、可认证、可复用的工程属性。

  • 可在真实机器人上执行
  • 确定性行为（在已声明约束内）
  • 显式前置条件 / 安全边界 / I/O Schema
  • 可通过证据与指标进行验证

为什么 RSS 要强调 Skill

物理执行的成本与风险远高于软件调用，因此必须引入验证、证据与治理机制。RSS 的目标是让技能可迁移、可信、可规模化复用。
